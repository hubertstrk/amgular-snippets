# responsive

| class | screen                        |
| ----- | ----------------------------- |
| `sm`  | small screens e.g. phones     |
| `md`  | medium screens e.g. tablets   |
| `lg`  | clarge screens e.g. notebooks |
| `xl`  | larger screens e.g monitors   |
